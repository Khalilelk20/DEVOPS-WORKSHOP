Atelier interactif 2 - Niveau intermédiaire

Titre du Workshop : Déploiement d’un site Web sur AWS avec Jenkins, Docker et Ansible

Description : Ce workshop vise à enseigner aux participants comment déployer un site web sur Amazon Web Services (AWS) en utilisant un pipeline d'intégration continue et de déploiement continu (CI/CD) automatisé. Nous utiliserons Jenkins comme outil d'intégration continue, Docker pour la conteneurisation de l'application, et Ansible pour l'orchestration et la gestion de l'infrastructure AWS.
